# Supply Chain Sales Analysis

## Project Overview
This project presents an end-to-end analysis of supply chain and sales data using Excel, SQL, Python, and Power BI. The objective was to transform raw data into actionable insights and support data-driven decision-making.

## Tools & Technologies
- **Excel** – Data cleaning and initial analysis  
- **SQL** – Advanced querying and data analysis (CTEs, window functions, aggregations)  
- **Python (Pandas, NumPy, Matplotlib)** – Data cleaning and exploratory data analysis (EDA)  
- **Power BI** – Data modeling, DAX measures, and interactive dashboards
- 
## Key Analysis Performed
- Data cleaning and transformation to ensure data accuracy  
- Revenue and profit analysis across markets and products  
- Customer segmentation (new vs. repeat customers)  
- Sales trends over time (daily and monthly)  
- Shipping performance and delay analysis  
- Market contribution to total revenue  

## Key Insights
- Identified top-performing products and regions contributing most to revenue  
- Analyzed customer behavior to distinguish repeat vs. new customers  
- Detected patterns in shipping delays impacting operational efficiency  
- Highlighted trends in sales performance over time
